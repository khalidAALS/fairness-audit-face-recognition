Help and User Guide

Learn what the system does, how to use the dashboard and plugin, and how to interpret the fairness metrics and mitigation results.


What this system does
This system is a fairness auditing dashboard and mitigation tool for classification systems. It evaluates performance differences across demographic groups, specifically race and age using metrics such as Accuracy, False Positive Rates (FPR), False Negative Rates (FNR), and disparity gaps.

How to use the dashboard
The dashboard pages display precomputed baseline and mitigated results. These charts help users compare subgroup performance and identify demographic disparities that may not be visible through overall accuracy alone.

How to use the plugin audit
The plugin page allows you to upload a CSV file containing classifier outputs. The system validates the file format, computes fairness metrics, and optionally performs threshold based mitigation if probability scores are included.

Required CSV columns
  • y_true - Ground truth label
  • y_pred - Binary predicted label
  • y_score - Model confidence or probability score (needed for mitigation)
  • race - Demographic race group
  • age_group - Demographic age category
You must provide y_true, race, and age_group, and either y_pred or y_score.

Understanding the metrics
  • Accuracy - Overall proportion of correct predictions
  • FPR - Rate of incorrectly predicting a positive outcome
  • FNR - Rate of incorrectly predicting a negative outcome
  • Disparity Gap - Difference between the highest and lowest subgroup metric values

How mitigation works
If a y_score is provided, the system tests multiple thresholds and selects the one that reduces demographic disparity while limiting overall performance degradation. This is a post processing method which means it changes the decision threshold rather than retraining the model.

Exporting charts
Each chart includes an export (Save) option in the top right Plotly toolbar. You can download charts as image files for reports, presentations, or further review.
