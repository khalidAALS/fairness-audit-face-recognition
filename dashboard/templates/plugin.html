Plugin Audit

Upload a CSV from any face recognition system (or classifier) to compute fairness metrics.

Upload CSV

Required columns: y_true, race, age_group + either y_pred or y_score.
If you include y_score (0–1), the plugin can run mitigation (threshold tuning).
Download sample CSV format: sample_plugin_input.csv
CSV Requirements
Required:
  • y_true (0/1)
  • race (e.g., White, Black, East Asian...)
  • age_group (e.g., 20-29, 30-39...)
  • y_pred (0/1) OR y_score (0–1)
Notes:
  • If only y_score is provided, baseline predictions are made with threshold 0.5.
  • Mitigation searches for a threshold that reduces demographic disparity (FPR/FNR gaps).
